## Installation

* clone the repo
* ```cd retro-rice```
* ```./installer.sh```
and let it cook

## Set Up
- go to wallpaper folder
  - change it to the one inside
- open panel
  - go to display tab
  - untick lock panel
  - drag the panel (taskbar in windows) to the top
  - tick lock panel
  - go to items tab
  - modify it to your liking
